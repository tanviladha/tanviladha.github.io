Personal Project: PrepBot

I developed PrepBot as a personal project to gain experience with natural language processing techniques and chatbots. I worked on this project in three phases. I used Python's difflib library in the first phase to create a basic version that saved user prompts in a file to learn over time. In the second phase, I incorporated natural language processing by feeding the bot articles from the web and employing a term frequency-inverse document frequency (TF-IDF) vectorizer and cosine similarity to generate responses. The final phase involved integrating OpenAI’s APIs, which produced the most effective and sophisticated version of the chatbot.

Personal Project: Credit Card Fraud Detector

Using a Kaggle dataset and leveraging my AI experience, I developed a logistic regression model to predict fraudulent credit card transactions. The dataset consists of anonymized transactions by European cardholders in 2023. For this project, I utilized Pandas and NumPy for data handling, Matplotlib and Seaborn for visualization, and Scikit-learn for modeling. I chose logistic regression due to its suitability for binary classification, using log loss as the loss function. I also performed hyperparameter tuning. Detailed findings are documented in the Jupyter Notebook available in the repository.
