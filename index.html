Break Through Tech AI @ UCLA

Break Through Tech AI is a program that begins with an AI/ML-focused curriculum and then advances to real-world applications under the guidance of industry mentors. I gained hands-on experience in data cleaning techniques (winsorization, null-value removal, one-hot encoding, feature transformations) using tools like Pandas, NumPy, Matplotlib, Seaborn, and SciPy. I developed machine learning models using methods such as k-nearest neighbors, decision trees, logistic and linear regression, ensemble methods (random forest, gradient boosted decision trees, stacking, bagging, boosting), neural networks, and natural language processing. Additionally, I tuned these models by applying hyperparameters, loss functions, gradient descent, and optimizers, and evaluation metrics.

Personal Project: PrepBot

I developed PrepBot as a personal project to gain experience with natural language processing techniques and chatbots. I worked on this project in three phases. I used Python's difflib library in the first phase to create a basic version that saved user prompts in a file to learn over time. In the second phase, I incorporated natural language processing by feeding the bot articles from the web and employing a term frequency-inverse document frequency (TF-IDF) vectorizer and cosine similarity to generate responses. The final phase involved integrating OpenAI’s APIs, which produced the most effective and sophisticated version of the chatbot.

Personal Project: Credit Card Fraud Detector

Using a Kaggle dataset and leveraging my AI experience, I developed a logistic regression model to predict fraudulent credit card transactions. The dataset consists of anonymized transactions by European cardholders in 2023, with features V1-V28 anonymized and the 'class' label indicating fraud (1) or not fraud (0). For this project, I utilized Pandas and NumPy for data handling, Matplotlib and Seaborn for visualization, and Scikit-learn for modeling. I chose logistic regression due to its suitability for binary classification, using log loss as the loss function. The model was trained using Scikit-learn's train-test split function and evaluated based on log loss and accuracy. I also performed hyperparameter tuning. Detailed findings are documented in the Jupyter Notebook available in the repository.
